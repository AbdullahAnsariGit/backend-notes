MongoDB Basic Guide with Schema & Methods

1. MongoDB Kya Hai?

MongoDB ek database hai jo data ko JSON-like documents me store karta hai. Ye bilkul ek digital notebook jaisa hai jisme tum apni information likh sakte ho aur baad me padh sakte ho. Isme data tables ke bajaye flexible documents me save hota hai.

Example: Ek student ka data aise dikhega: 

{
    "name": "Ali",
    "age": 12,
    "class": 5
}

2. MongoDB Atlas Kya Hai?

MongoDB Atlas ek cloud service hai jisme tum apna MongoDB database online host kar sakte ho. Tumhe apne computer pe kuch install nahi karna, bas internet se connect karke use karo. Ye ek online storage jaisa hai jo sab kuch manage karta hai.

Fayda: Atlas backup, security, aur scaling khud handle karta hai.

3. Cluster Kya Hota Hai?

Cluster ek group of servers hota hai jo ek saath kaam karte hain tumhara database chalane ke liye. Atlas me jab tum database banate ho, to ek cluster banta hai jisme data safe rehta hai.

Simple Example: Cluster ek bada box hai jisme tumhara data store hota hai, aur kai computers milke isko chalate hain.

4. Schema Kya Hai?

Schema ek blueprint hai jo batata hai ki tumhara data MongoDB me kaisa dikhega. Ye define karta hai ki tumhare document me kaun-kaun se fields honge aur unka type kya hoga (jaise string, number, etc.).

Example: Ek student ka schema aisa ho sakta hai: 

const studentSchema = new mongoose.Schema({
    name: String,
    age: Number,
    class: Number
});

Yaha name ek string hoga, age aur class numbers honge.

5. Model Kya Hai?

Model ek class hai jo schema ke basis pe banaya jata hai. Ye tumhe allow karta hai database me data save, read, update, ya delete karne ke liye. Model ke through tum apne data ke saath kaam kar sakte ho.

Example: Schema ke baad model banane ka code: 

const Student = mongoose.model('Student', studentSchema);

Yaha Student model hai jo studentSchema use karta hai. Ab tum Student se data add ya find kar sakte ho.

6. Methods Kya Hote Hain?

Methods ek tarika hai apne model me custom functions add karne ka. Ye functions tumhare data ke saath specific kaam kar sakte hain. For example, ek method bana sakte ho jo student ka full details return kare.

Example: Ek custom method banane ka code: 

studentSchema.methods.getDetails = function() {
    return `Name: ${this.name}, Age: ${this.age}, Class: ${this.class}`;
};

Is method se tum student ke details ek string me le sakte ho.

7. MongoDB se Connection Kaise Banaye?

MongoDB Atlas se connect karne ke liye connection string chahiye. Ye Atlas se milti hai jab tum cluster banate ho. Hum Mongoose use karenge kyunki ye schemas aur models ke saath kaam karna aasan banata hai.

Step-by-Step Connection Guide

  1. MongoDB Atlas pe Account Banaye:
    • Jao MongoDB Atlas pe aur sign up karo.
    • Ek free cluster banao (M0 free tier kaafi hai).
    • Connect button pe click karke Connect your application select karo. Connection string copy karo: 
      mongodb+srv://:@cluster0.mongodb.net/myDatabase?retryWrites=true&w=majority
  2. Node.js Project Setup Karo:
    • Ek folder banao aur terminal me jao.
    • Run karo: npm init -y (package.json file ban jayegi).
  3. Packages Install Karo:

    Ye packages chahiye:

    • mongoose: MongoDB ke saath structured way me kaam karne ke liye.
    • express: Web server ke liye.

    Terminal me run karo:

    npm install mongoose express
  4. Connection Code Likho:

    Ek file banao index.js aur ye code likho:

    const mongoose = require('mongoose');
const express = require('express');
const app = express();

// Connection String
const uri = "mongodb+srv://:@cluster0.mongodb.net/schoolDB?retryWrites=true&w=majority";

// Connect to MongoDB
mongoose.connect(uri, {
    useNewUrlParser: true,
    useUnifiedTopology: true
}).then(() => {
    console.log("Connected to MongoDB!");
}).catch(err => {
    console.error("Connection failed:", err);
});

// Schema
const studentSchema = new mongoose.Schema({
    name: String,
    age: Number,
    class: Number
});

// Method
studentSchema.methods.getDetails = function() {
    return `Name: ${this.name}, Age: ${this.age}, Class: ${this.class}`;
};

// Model
const Student = mongoose.model('Student', studentSchema);

// Example: Add a Student
async function addStudent() {
    const student = new Student({
        name: "Ali",
        age: 12,
        class: 5
    });
    await student.save();
    console.log("Student Added:", student);
    console.log(student.getDetails()); // Use custom method
}

addStudent();

// Start Server
app.listen(3000, () => {
    console.log("Server running on port 3000");
});

    Note: aur ko apne Atlas credentials se replace karo. schoolDB database ka naam hai.

  5. Run Karo:

    Terminal me ye command run karo:

    node index.js

    Agar "Connected to MongoDB!" aur student details print hote hain, to sab kaam kar raha hai!

8. Bonus: Data Ke Saath Kaam Karna

Yaha kuch basic operations hain jo tum model ke saath kar sakte ho:

  • Add Data: Upar wale code me student.save() se data add kiya.
  • Find Data: 
    const students = await Student.find({ class: 5 });
console.log("Class 5 Students:", students);
  • Update Data: 
    await Student.updateOne({ name: "Ali" }, { age: 13 });
console.log("Student Updated!");
  • Delete Data: 
    await Student.deleteOne({ name: "Ali" });
console.log("Student Deleted!");

9. Recommended Packages

  • mongoose: Schemas, models, aur methods ke liye best hai.
  • express: Web apps banane ke liye.
  • mongodb: Agar schemas nahi chahiye, to direct MongoDB driver use karo.

10. MongoDB ke Fayde

  • JSON-like format se data fast aur flexible hai.
  • Atlas se cloud me aasani se kaam kar sakte ho.
  • Mongoose ke saath schemas aur methods banakar code organized hota hai.

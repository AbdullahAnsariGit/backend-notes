Web Application Architecture

  • Users: Data collect aur results dekhte hain.
  • Frontend: HTML, CSS, JavaScript se banaya jata hai (jo users dekhte hain).
  • Web Server: Node + Express requests handle karta hai.
  • Backend: File system (files store karta hai) aur database (data manage karta hai) se kaam hota hai.

File System

  • Files (HTML, CSS, images) ko store aur organize karta hai.
  • Example: "public/images/logo.png" ko Express ke saath serve karna (app.use(express.static('public'));) aur React mein se use karna.

MERN Stack

  • React: Frontend ke liye (UI banata hai).
  • Node + Express: Backend aur web server kaam (requests handle karte hain).
  • MongoDB: Data store karta hai (NoSQL).
